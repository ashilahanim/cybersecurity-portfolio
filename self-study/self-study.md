## 24/12/2025
- Topic: Network Addresses Translation (NAT)
- What is it: Translate private IP addresses to public IP addresses from a local network/device and translate public IP addresses to private from the internet to a local / private devices.
- Why it matters: Protect private network, conserves public IPv4 addresses, let private network to connect to internet
- Tool/protocol: Routers, firewall, gateway device
- One example: A person trying to protect herself by using pseudonym.
